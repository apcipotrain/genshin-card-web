# 原神杀（Genshin Card）使用文档

> 版本：v2.0 | 日期：2026-06-18

---

## 一、环境要求

- **Node.js** >= 18.0
- **npm** >= 9.0
- 现代浏览器（Chrome / Edge / Firefox 最新版）

---

## 二、快速启动

### 2.1 安装依赖

```bash
cd genshin-card-web
npm install
```

> 提示：如果下载慢，先切换国内镜像：
> ```bash
> npm config set registry https://registry.npmmirror.com
> ```

### 2.2 启动开发服务器（含PVP服务端）

```bash
npm start
```

此命令会同时启动：
- PVP 服务端（`npx tsx server/index.ts`，端口 3457）
- Vite 开发服务器（端口 3000，自动代理 Socket.IO 到 3457）

浏览器打开 `http://localhost:3000`

### 2.3 仅启动前端（PVE 单机模式）

```bash
npm run dev
```

> 注意：PVE 模式不需要服务端，但经验值上报和账号登录需要服务端在线。

### 2.4 仅启动服务端

```bash
npm run server
```

### 2.5 构建生产版本

```bash
npm run build
```

产物在 `dist/` 目录。

## 三、账号系统

### 3.1 注册/登录

- 首次使用需注册账号（名称 + 密码）
- 登录后 token 保存在 `sessionStorage`，刷新页面自动认证
- 账号数据持久化在 `server/accounts.json`

### 3.2 账号管理

- 主页面左侧栏点击账号头像 → 弹出账号管理面板
- 显示：昵称、等级、经验进度（如 43/200）
- 可修改昵称（不超过12字符）

### 3.3 经验值

- `exp` 为累计经验总量（从 Lv.1 开始累计）
- 面板显示 `currentLevelExp / nextLevelExp`（当前等级内经验 / 升级所需经验）
- PVP 结算后自动增加经验，升级时显示升级动画
- PVE 结算后通过 `add_exp` 事件上报服务器

## 四、页面导航

### 4.1 主页面
- **PVE 闯关模式**（左）：点击进入关卡选择
- **PVP 联机模式**（右）：点击进入匹配界面
- **左侧栏**：账号管理、成就系统、📖图鉴、设置

### 4.2 PVE 关卡选择
- 9章横向滚动（蒙德→璃月→稻妻→须弥→枫丹→纳塔→挪德卡莱→至冬→深境螺旋）
- 点击箭头或触摸滑动切换章节
- 点击章节卡片进入关卡弹窗
- 当前已实现：蒙德7关 + 璃月12关
- 解锁条件：等级 + 前一章节累计20星
- 点击解锁的关卡 → 进入选将 → 进入游戏

### 4.3 PVP 匹配
- 左侧：已有房间列表（可滚动）
- 右侧：创建房间面板
  - 输入房间名称
  - 开启/关闭私密模式（设置密码）
  - 创建房间 → 进入等待界面
- 点击公开房间直接加入，私密房间需要输入密码
- 快速匹配：自动加入第一个有空位的公开房间

### 4.4 等待界面
- 显示8个玩家位（4x2网格）
- 自己的位置用金色边框+👑标记
- 房主可设置 AI 填充数量
- 点击"准备"按钮准备/取消准备
- 房主点击"开始选将"启动游戏

### 4.5 选将流程
- **主公选将**：从3神+3非神中选1
- **非主公选将**：从3个随机候选中选1
- 选将倒计时30秒，超时自动选择
- AI 主公/AI 玩家由服务器自动选将

### 4.6 大厅图鉴系统
- **入口**：大厅左侧栏点击 📖 图鉴
- **排序**：按国家分组（蒙德→璃月→稻妻→须弥→枫丹→纳塔→挪德卡莱）
  - 每个国家内：七神排最前，其余武将按名称首字母排序
- **英雄卡片**：圆形头像 + ★神标签 + 姓名 + 称号 + 元素/血量
- **详情弹窗**：点击任意英雄 → 弹出完整资料（称号、区域、元素、性别、体力上限、技能描述）

## 五、游戏界面

### 5.1 战场布局（逆时针排列）
- **上方3人**：从左到右为逆时针远→近
- **左侧2人**：从上到下
- **右侧2人**：从下到上
- **下方自己**：人类玩家

### 5.2 座位卡片信息
- 元素图标（🍃🪨⚡🌿💧🔥❄️）
- 名字 + 身份（主公/忠臣/反贼/内奸，PVE无身份）
- HP 血条（绿/橙/红）+ 数值
- 装备图标（⚔武器 🛡防具 🐴马）
- 手牌数量
- 铁索连环标记 ⛓️
- 判定区内容
- 当前回合武将外框旋转金色光晕

### 5.3 信息栏
- **左上**：⏱ 游戏计时器（MM:SS 格式） + 牌堆剩余数量
- **中间**：当前阶段（准备/判定/摸牌/出牌/弃牌/回合结束）
- **右上**：第X轮 · 第X回合

### 5.4 右侧面板
- **上方**：游戏记录（实时滚动日志，底部时自动滚动，上翻时不强制）
- **下方**：聊天面板（预留）

### 5.5 游戏交互
人类玩家回合中，会弹出交互窗口：
- **出牌阶段**：选择手牌打出，或点击"结束出牌"
- **选择目标**：选择杀/锦囊的目标玩家
- **响应索要**：出杀/闪/桃/酒/无懈可击等
- **弃牌阶段**：手牌超过体力上限时选择弃牌
- **装备/武器效果**：是否发动装备技能
- **铁索连环**：选择"重铸"还是"连环"
- **五谷丰登**：从桌面牌中选择一张
- **火攻**：展示手牌后选择目标
- **雌雄双股剑**：选择"弃牌"或"摸牌"
- **丈八蛇矛**：选择两张牌合成杀

### 5.6 动画效果
- **打出/弃置牌**：显示实际卡牌PNG图片（明牌），从座位飞到中央
- **摸牌**：显示卡背动画（暗牌），从中央飞入对应位置
- **黄光连线**：杀/锦囊指定目标时，source→target 光束动画
- **其他玩家操作**：在对应座位处显示明牌/暗牌弹出动画

### 5.7 设置面板
点击游戏界面齿轮图标打开设置：
- **音乐开关**：开启/关闭背景音乐
- **音量滑块**：0-100% 音量调节
- **AI出牌时间**：0.3秒（极速）～ 5.0秒（极慢），默认1.6秒
- **BGM/背景自动切换**：根据主公/章节所属国家自动切换对应BGM和背景图

### 5.8 卡牌效果
游戏使用三国杀标准160张牌堆，所有卡牌效果均已实现：
- 基本牌：杀（含火杀/雷杀）、闪、桃、酒
- 锦囊：南蛮入侵、万箭齐发、桃园结义、五谷丰登、无中生有、过河拆桥、顺手牵羊、决斗、火攻、借刀杀人、铁索连环、无懈可击
- 延时锦囊：乐不思蜀、兵粮寸断、闪电
- 装备：诸葛连弩、青釭剑、丈八蛇矛、贯石斧、青龙偃月刀、方天画戟、寒冰剑、古锭刀、朱雀羽扇、麒麟弓、雌雄双股剑、八卦阵、仁王盾、藤甲、白银狮子、+1马、-1马 等

## 六、结算界面

### 6.1 PVP 结算
- 显示胜利方
- 列出所有玩家的昵称、身份、武将、体力、存活状态
- 显示获得经验（基础经验 + 击杀加成）
- 升级时显示升级动画
- 可关闭弹窗继续查看战场
- 点击"返回主页"回到主页面

- 星级评定：★★★（全员存活）/ ★★（阵亡1人）/ ★（阵亡2人）
- 显示获得经验
- 点击"返回章节"回到关卡选择

## 七、常见问题

### Q: 游戏太慢/太快？
A: 打开设置面板（齿轮图标），在"AI出牌时间"下拉框中选择：0.3秒（极速）～ 5.0秒（极慢），默认1.6秒。

### Q: 为什么角色没有技能？
A: v2.0 版本已有 **29 位武将** 拥有完整技能实现（七神7 + 非神22），其余武将将在后续版本实现。

### Q: PVP 联机可以用吗？
A: v2.0 已实现完整的 PVP 联机功能，包括账号系统、房间管理、选将、实时对战、断线重连、经验结算。使用 `npm start` 同时启动前端和服务端。

### Q: 断线了怎么办？
A: Socket.IO 自动重连。游戏中断线的真人玩家会被 AI 暂时接替，重连后恢复控制。逃跑玩家结算时经验为0。

### Q: 如何添加新角色？
A: 修改 `src/data/heroes.ts` 中的 `HEROES` 数组，添加角色信息（名称/国家/元素/性别），并在 `SkillManager.ts` 中实现技能。

### Q: 如何修改卡牌数据？
A: 修改 `src/data/CardData.ts` 中的 `CARD_DATA` 数组。

### Q: 如何添加新的PVE关卡？
A: 修改 `src/data/PVELevels.ts`，在对应章节的 `LEVELS` 数组中添加关卡数据。
