### 技能实现架构

| 角色 | 技能 | 实现方式 | 系统集成点 |
|------|------|---------|-----------|
| 林尼 | 魔术 | 主动技：选牌→全场角色猜基本/非基本→猜对回血/猜错扣血→摸X张→交牌 | executeActiveSkill + applyHealthLoss + HpChanged |
| | 奇迹 | 被动技：手牌首次为0→摸至体力上限 | onAfterCardPlay + checkLyneyMiracle |
| 娜维娅 | 刺玫 | 锁定技：受到伤害恒为1 | DamageSystem.naviaRoseHandler |
| | 说服 | 主动技：选模式→效果1给牌禁杀/效果2拿牌禁目标 | executeActiveSkill + resetTurnFlags |
| 克洛琳德 | 决斗 | 主动技：摸1张牌→本回合该点数及以上可当决斗 | executeActiveSkill + 点数阈值标记 |
| | 剧团 | 主动技：限一次禁用全场角色技能直到下回合/阵亡 | executeActiveSkill + SkillManager._clorindeTroupeActive |
| 希格雯 | 护士 | 主动技：弃桃→令其他角色回复1点体力 | executeActiveSkill |
| | 温度 | 锁定技：打出桃回复量+2 | CardEffectManager.executePeachLogic |

### 系统级改动

**克洛琳德-剧团 全场禁技系统**：
- `SkillManager._clorindeTroupeActive` 全局标志
- `getSkills()` 返回空数组（按钮不可见）
- 下回合开始或克洛琳德阵亡时自动解除
- 不影响已存在的状态（标记、buff等）

### 3.2 新增 prompt 类型

| 方法 | 用途 | 英雄 |
|------|------|------|
