## 三、核心系统修复

### 3.1 铁索连环
- **修复**：`CardEffectManager.ts` 传导循环新增 `if (otherPlayer.isDead) continue;`

### 3.2 判定动画补全
- **歌颂/罪舞**：`SkillManager.ts` 翻牌后发出 `GameEvent.JudgeResult`
- **妮露花舞**：每张判定牌发出 `JudgeResult`
- **那维莱特龙权**：所有判定来源全覆盖

### 3.3 弃牌动画
- **根因**：`DeckManager.sendToDiscard` 未传 `playerId`
- **修复**：清空 `card.cardSource` 前提取 `sourcePlayerId` 发出

### 3.4 经验值与星数实时更新
- **星数**：`ChaptersPage.ts` 新增 `refreshStars()`，`show()` 时重新读取显示
- **经验**：`GamePage.ts` PVE 结算时同步 `socketManager.setAccount()`

### 3.5 PVE 座位还原
- `getPVESeatLayout` 恢复为 `[0,1,2,3,4,5,6,7]` 序列

### 3.6 PVE 敌友判定
- `AIDriver.ts` `isEnemy()` / `GamePage.ts` `isHostile()`：PVE 基于 `faction`，PVP 基于 `role`

### 3.7 PVE 失败显示
- `GamePage.ts` `showResultModal`：敌方胜 → 红色 "💀 挑战失败"

### 3.8 无懈可击读条优化
- `promptNullification` 超时：12秒 → 5秒
- `renderGraceNullifySection` 超时：8秒 → 5秒
- 提示文字同步更新（含五谷弹窗内嵌）
- "放弃打出"按钮标注5秒超时

### 3.9 火免疫（宵宫琉金）
- **重大bug修复**：`isImmuneToFire` 之前从未被 `DamageSystem` 调用
- 火属性伤害前新增检查，直接 `return` 不受伤
